# Change Log

All notable changes to the CodeCollab extension will be documented in this file.

## [0.1.0] - Initial Release

### Added
- Real-time collaborative editing using Yjs CRDT
- Multi-cursor support with user names and colors
- Video and audio communication via WebRTC
- Voice comments on code lines
- AI assistant integration with Ollama
- Shared terminal functionality
- Session management (create, join, leave)
- Permission levels (read, edit, admin)
- Brutalist dark theme for UI

### Technical
- WebRTC peer-to-peer mesh architecture
- Socket.io signaling server
- Support for 2-8 simultaneous participants
- Free STUN servers for NAT traversal
